Harap file ini dibuka terlebih dahulu 

Petunjuk Pemakaian :

1. Pastikan file sudah terunduh

2. Pastikan membuka folder dan file menggunakan VSCode sebagai Integrated Development Environment(IDE), kalau tidak menggunakan VSCode, saya tidak bertanggung jawab ^^

3. Pastikan memiliki koneksi internet, jika tidak ada sebenarnya tidak mempengaruhi program chatbotnya, hanya tidak bisa mengeluarkan suara dan motion dari character itu sendiri

4. Buka terminal pada VSCode dan ketik "python ai.py"

5. Menunggu kurang lebih 10 detik

6.  Nanti akan keluar, pada tampilan yang dibawah 

* Serving Flask app 'ai'
 * Debug mode: on
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on http://127.0.0.1:5000
Press CTRL+C to quit
 * Restarting with stat
 * Debugger is active!
 * Debugger PIN: 683-033-120 

7. Copy bagian ini "http://127.0.0.1:5000", lalu tempelkan ke browser seperti google chrome

8. Web Chatbot akan muncul setelah menempelkan browser 

9. Kamu bisa bertanya-tanya seputar apapun itu, entah seputar pemrograman, matematika, anime, teman virtual, tanya-jawab, seperti layaknya chatgpt tetapi web ini jauh untuk dikatakan bagus

10. Jika ingin mengetik chat, harus menggunakan Bahasa inggris karena voice dari character tersebut menggunakan Bahasa inggris, jika tetap menggunakan bahasa selain Bahasa inggris, maka logatnya cukup kurang bagus untuk didengar.

11. Hal yang sama juga berlaku untuk mengaktifkan mic, perlu menggunakan Bahasa inggris

12. Untuk melakukan reset history, maka restart saja browsernya akan otomatis history chatnya hilang

13. Jika sudah selesai menggunakan chatbot tersebut bisa tutup saja browsernya dan lakukan di terminal dengan perintah CTRL+C untuk keluar 

14. Saya harap dapat mengerti cara penggunaan chatbot ini ^^
